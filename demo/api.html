gFrame : start | stop

  • gFrame.stop()

gFrame : event

  • gFrame.ready(function, global)
  • gFrame.load(function, global)
  • gFrame.unload(function, global)
  • gFrame.trigger(type)
  • gFrame.removeEvent(type, func)

gLayer : create | remove

  • gFrame(id_string, code_string, option_object)
  • gFrame.create(id_string, code_string, option_object)
  • gFrame.remove(id)
  • gFrame.get(id) - It returns gLayer
  • gFrame(id).remove()
  • gFrame.has(id)
  • gFrame.size()
  • gFrame.list()
Don't worry about creating same id of gLayer. Same id is ignored.
Use 'gFrame.has' for checking existence.

gLayer : dimension

  • gLayer.top([val])
  • gLayer.bottom([val])
  • gLayer.left([val])
  • gLayer.right([val])
  • gLayer.width([val])
  • gLayer.height([val])
  • gLayer.zIndex([val])
It returns gLayer instance or value.

gLayer : etc

  • gLayer.text(value) - innerHTML
  • gLayer.style(value)
  • gLayer.opacity(value)

gFrame : once

  • gFrame.once(id, code or function)
  • gFrame.once.remove(id)
  • gFrame.once.empty()
  • gFrame.once.size()
  • gFrame.once.list()
If you excute small code written in the document. Once is useful.
